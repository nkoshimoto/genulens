# Citation Notes

The authoritative citation list is the one printed by the PARSEC/CMD service
and by each downloaded CMD table. This file records the references relevant to
the current `requests.json` settings so they are not lost when the web service
temporary output URLs expire.

Data source:

- PARSEC/CMD web interface, `https://stev.oapd.inaf.it/cgi-bin/cmd`

Acquisition helper:

- `ezpadova`, `https://github.com/mfouesneau/ezpadova`
